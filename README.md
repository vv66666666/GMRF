# GMRF

This project implements a detection framework that combines user behavioral signals, preference-aware representations, graph-based relation learning, and probabilistic inference.
Experiments are conducted on public datasets, including: AmazonCN and YelpZip. Please prepare the input data according to your local setup.
Experiments are conducted under different labeled-data ratios:5% 10% 30% 50%. Under each setting, a subset of labeled users is randomly sampled for training, and the remaining users are used for inference and evaluation. Each setting is repeated multiple times and the average results are reported.
